# 🪐 Solar System 3D

A dynamic 3D visualization of the solar system built with **Three.js**. 
*Uma visualização 3D dinâmica do sistema solar construída com Three.js.*

## 🚀 Technologies | Tecnologias
* **JavaScript** (ES6+)
* **Three.js** (3D Engine)
* **HTML5 / CSS3**

## 🛠️ How to run | Como executar
1. Clone this repository | *Clone este repositório*
2. Open `solar-system.html` in your browser | *Abra o arquivo `solar-system.html` no seu navegador*
